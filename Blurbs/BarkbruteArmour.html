This cocoa-colored body armor is easy to make. You need 2 Barkbrute Skins and a Knife. You need to be in Jungle terrain (so that you can find things to tie and bind it). Issue the MAKE order: M 107. It takes 18 Action Points. Barkbrute Armor provides Armor Class 3 protection. You can carry one suit.
