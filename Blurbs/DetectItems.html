When using this spell, you'll search the square you occupy for other Monsters. Each time you find one, you'll cast this spell. This spell reveals all of the items the Monster encountered is carrying. Issue the CAST SPELL order: C 66. It takes 32 Action Points and uses 24 Spell Points.

Note: this spell does not detect monsters at Far Structures, inside dark places, etc.
