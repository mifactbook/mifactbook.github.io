Ruck A resides at the Crooked Nose. Ruk B resides at the Black Swan. Ruk A desires Item C from Ruk B requiring lengthy teleports and travel resulting in old, tired Ruks. Fear not, the Zoingot Wizards have learned the secret of making a one-way Zip Tunnel to carry objects from one place to another. For a Ruk it can be a simple though tedious process to use these same magics. Therefore if you should want to send an item and make sure it gets there most of the time, just use the TELEPORT ITEM SPELL. Because of the mystical energies required the spell must occur in two phases.

  • Phase I: use the order C 320 [Item#]
  • Phase II: use C 320 [Monster#]

Each phase costs 50 Action Points and 50 Spell Points.
